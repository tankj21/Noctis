# Commands

## slot

### play <bet>

スロットマシンをプレイできます。

### coins

所持コインと統計の確認ができます。

### jackpot

現在のジャックポット情報を表示します。

### ranking

コインランキングを表示します。

### help

スロットの使い方を表示します
